# 产品信息管理系统

## 一.介绍
基于PyQt5和MySQL数据开发的产品信息管理系统

## 二.安装
### 1.客户端安装
python -m venv VENV

VENV\Scripts\activate.bat

pip install -r requirement.txt
### 2.服务器安装
安装mySQL，创建账户
### 3.使用
用mySQL账户登陆
